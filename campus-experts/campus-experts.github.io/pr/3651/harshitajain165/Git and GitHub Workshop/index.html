Git and GitHub Workshop

The Git and GitHub Workshop serves as a basic introduction for beginners to git, common commands and GitHub. The workshop is a part of the pre-events being organized for Break the Barrier hackathon by the student community of LDRP Institute of Technology and Research. This will be followed by a hands-on demonstration of the same and an in-depth discussion about the GitHub Student Developer Pack. How to avail the same shall be discussed and an overview of the GitHub Campus Experts Program will be given.

Date: 2022/08/07

Time: 3 p.m. IST onwards
