Event Dedicated URLs

The event dedicated URLs are a specialized feature in the Student Developer Pack designed to expedite the access to the pack‚Äôs benefits for event participants. It is a feature that is only available to GitHub Staff and GitHub Campus Experts. Unlike the standard access procedure, these URLs bypass the typical three-day cooldown period, granting immediate access to the tools and resources contained within the pack while also provide preference for those who apply using it.

It is important to share this URL when organizing hackathons, workshop or activities that rely on the Student Developer Pack. You want to avoid attendees waiting for their request to be approved and access granted.

How to create a URL

  1. When you create an event issue on the being-an-expert repositoryopen in new window, you will be asked to fill the URL template.
  2. Fill the template URL with the event information: https://education.github.com/benefits?utm_source=yyyy-mm-dd-eventname. Avoid using any characters that might modify where the URL is pointing (Ex. /, @, .)
  3. Once you have created the URL and posted the event issue. The GitHub Education team will review and approve the URL so attendees get all the benefits faster.
