OPEN COLLECTIVE: For all events until further notice:

As of December 1, 2024, All reimbursements will be done through Open Collective until further notice.

Steps to request reimbursements via Open Collective

  • Head over to https://opencollective.com/github-education and follow the GitHub Education Collective.
  • After following, find the "Submit Expense" button.
  • Next, select "Reimbursement" (For CE Events) from the two options. After doing so, fill out the Payee information and add your bank details if you don't have a payout method already.
  • Once through, click "Next" and then fill out Expense details. Add the tag "ce-sponsorship" below "Tag your expense".
  • Make sure you upload an invoice with the following address: Open Source Collective 440 N. Barranca Ave #3939 Covina, CA 91723
  • Invoices paid via OC need to have the aforementioned address in the Bill To section.
  • Remainder is straightforward, fill out the "Invoice details" section based on the doc you're submitting, add notes if any and you're all set to submit!

Filling out the invoice

  • Invoice templateopen in new window Make a copy of this and name it YourLastName_invoice_##### where ##### is the invoice number (see below for Invoice number instructions).
  • Create an invoice for the total amount, this invoice should include a list of all expenses and their amounts.

  • For the Invoice number, this indicates how many invoices you have sent the program and it should start with your GitHub IDopen in new window. After your GitHub Id, please add a 001 to indicate which invoice number this is for you. The next time you submit an invoice it should be "002" and so on and so forth. For example, if Pj were to create an invoice for the first time, it would be numbered 65838556001. Pj's GitHub ID is 65838556.
  • The line items in the invoice do not need to say "Dates Worked." Rather, it should be formatted to say what the line item is for, and a total cost for that particular line item.
  • To get the total cost in the Invoice, work backwards from the total cost. If the pizza was $250, and the event was 5 hours, then the hourly rate should be $50. Inputting the hourly rate and the hours will automatically fill out the total at the end of the row.
  • Invoices are paid in 30 days, sometimes longer for international students.
